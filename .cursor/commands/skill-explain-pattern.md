# Explain Pattern

In-place explanation of a specific idiom, function, decorator, or hook.

## What I Need From You

- Target: file + line range, or a pasted snippet.
- Optional narrower question (e.g. "why `@with_api_logging`?").

## What Happens Next

1. Identifies the idiom category.
2. Searches rules and ADRs for governing guidance.
3. Returns a terse explanation with citations and another-usage reference.

## Tips

- Use before copying a pattern into new code.
- Pair with `/skill-check-conventions` if the pattern looks off.
- Ask targeted questions; the skill is focused, not exploratory.
