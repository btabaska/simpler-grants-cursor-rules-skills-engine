# Review PR

Perform a comprehensive code review following simpler-grants-gov standards.

## What I Need From You

Provide ONE of:
- **PR number:** "#1234"
- **Branch name:** "feature/new-search-ui"
- **File list:** "I changed these files: [list]"

Or just say "review my current changes" and the agent will check your working tree.

## What Happens Next

The PR Review Skill will:
1. Identify all changed files and map them to applicable project rules
2. Run the 8-section review checklist
3. Invoke Compound Engineering specialists for each domain
4. Flag issues by severity (must-fix, should-fix, nit, praise)
5. Produce a review in Brandon's voice/style

## Tips for Better Results
- Mention if this is a draft PR (lighter review) or ready for merge (full review)
- Mention any areas you're specifically unsure about
- If this is a large PR, mention the key changes you want focused review on
