# Pattern Catalog

Browse a curated catalog of simpler-grants-gov implementation patterns with anti-pattern vs. correct-pattern code examples sourced from real files.

## What I Need From You

One of:

1. **A pattern name** — `api-route-thin`, `server-component`, `soft-delete-query`
2. **A keyword** — "route", "fetch", "form validation"
3. **A layer** — "api", "frontend", "forms", "database"

## What Happens Next

The Pattern Catalog agent will:
1. Resolve the request to a catalog entry
2. Quote a real anti-pattern (with file path and line range)
3. Quote the real correct pattern (with file path and line range)
4. Highlight the key differences
5. Cite the governing `.cursor/rules/*.mdc`
6. Suggest 2 adjacent patterns

This agent is READ-ONLY. All snippets come from real files; synthesized examples are clearly labeled.

## Tips for Better Results
- Ask for one pattern per invocation
- For convention text only, use `/convention-quick-lookup`
- For full request walkthroughs, use `/interactive-codebase-tour`
