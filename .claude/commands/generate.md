# Generate

Generate code following simpler-grants-gov patterns and conventions.

## What I Need From You

Describe what you want generated:

- **What type of code?** — route handler, service function, React component, database model, test, etc.
- **What should it do?** — the behavior or functionality
- **Where should it go?** — the domain or area of the application

## What Happens Next

The Code Generation Agent will:
1. Determine the correct domain and load applicable rules
2. Generate code following all project conventions
3. Run quality gate validation (conventions, language quality, domain-specific checks)
4. Present the code with a compliance checklist

## Tips for Better Results
- Mention an existing file to use as a pattern reference
- Be specific about fields, types, and behaviors
- Mention if this connects to existing code
