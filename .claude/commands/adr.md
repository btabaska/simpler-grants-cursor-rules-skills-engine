# ADR

Write an Architecture Decision Record for the simpler-grants-gov project.

## What I Need From You

Describe the decision:

1. **What decision needs to be made?** — "Which search engine to use" / "How to handle auth token refresh"
2. **Why now?** — What prompted this decision?
3. **Any constraints?** — FedRAMP, accessibility, open-source licensing, etc.

## What Happens Next

The ADR Agent will:
1. Load existing ADRs and architectural context
2. Draft an ADR following the project template (Status, Context, Decision, Alternatives, Consequences)
3. Reference relevant project constraints and prior decisions
4. Validate with architecture and convention specialists

## Tips for Better Results
- Mention alternatives you've already considered
- Note any constraints specific to this decision
- Reference related ADRs if you know of them
