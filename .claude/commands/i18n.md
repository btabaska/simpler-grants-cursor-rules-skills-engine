# i18n

Add or modify translations in the simpler-grants-gov frontend.

## What I Need From You

Describe the text you need:

1. **What page/feature?** — "Opportunities search page" / "Grant application form"
2. **What text?** — the actual user-facing strings
3. **Context** — where in the UI does this text appear?

## What Happens Next

The i18n Agent will:
1. Find or create the correct namespace in `frontend/src/i18n/messages/en/index.ts`
2. Add properly structured translation keys (PascalCase namespace, camelCase keys)
3. Update the component to use `useTranslations()` hook
4. Validate against i18n conventions

## Tips for Better Results
- Mention the component where the text will be used
- Group related strings together (e.g., all form labels for one section)
- For long content, break it into separate keys (descriptionP1, descriptionP2)
